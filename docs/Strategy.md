# Stratégie SaaS & Monétisation - Job Shift Tracker

## 1. Proposition de Valeur
**Pour les Recruteurs** : Accès exclusif et immédiat à des postes stratégiques qui viennent de se libérer. Ne plus attendre l'annonce, mais contacter le recruteur de l'entreprise qui perd un talent.
**Pour les Candidats** : Une plateforme fluide pour déclarer leur succès et aider la communauté à identifier les opportunités.

## 2. Plan MVP (Minimum Viable Product)
- **Cible** : Recruteurs en ESN et Cabinets de Chasse.
- **Fonctionnalités clés** :
  - Formulaire de déclaration simple (avec auto-complétion IA).
  - Dashboard avec flux temps réel.
  - Alertes emails immédiates pour les postes "Senior" ou "Management".
- **Canal d'Acquisition** : LinkedIn Automation (bot qui commente les posts de "New Position" pour inviter à déclarer le changement).

## 3. Monétisation
### Modèle Freemium / Subscription
- **Free** : Accès au flux avec un délai de 24h. Notifications limitées.
- **Premium (Recruteur Pro)** : 99€/mois. Accès temps réel (WebSocket), filtres avancés (technos, secteur), alertes SMS/Slack illimitées.
- **Enterprise** : API Access pour intégration directe dans les ATS (Workable, Lever, etc.).

## 4. Gamification (Incentive Utilisateurs)
Pour pousser les utilisateurs à déclarer leur poste :
- **Badges "Communautaires"** sur LinkedIn.
- **Système de parrainage** : "Déclare ton poste et débloque un guide exclusif sur la négociation de salaire".
- **Anonymat Garanti** : Possibilité de déclarer anonymement pour les postes sensibles.

## 5. Roadmap Scaling
1. **Phase 1** : Lancement manuel sur une niche (ex: Tech à Paris).
2. **Phase 2** : Déploiement du scraper LinkedIn global pour automatiser 90% des saisies.
3. **Phase 3** : Intégration IA pour prédire les départs (Analytics de turnover prédictif).
